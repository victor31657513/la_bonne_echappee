# La Bonne Échappée

This project displays a 3D visualization of a cycling route using Three.js. All third‑party libraries are bundled locally, including **togeojson**, to avoid cross‑origin errors when the site is hosted.

Open `index.html` in a modern browser to run the demo. If you encounter a 404 when requesting `togeojson.esm.js` from a CDN, use the bundled file `js/togeojson.es.mjs` instead.
